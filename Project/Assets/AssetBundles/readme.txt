The assetbundle folder contains the useful files that can be transfered into Animation Prep Studio. 
This folder contains uniquely named UUID folders corrosponding to the prefab it was built from.
It stores files such as a compressed .blend file of the original model, thumbnails and the assetbundle it's self.

Each folder may be copied to "VR_MocapAssets" folder in the application's presistent storage directory:

%USERPROFILE%\LocalLow\Animation Prep Studios\AnimationPrepStudio_Lite\VR_MocapAssets
